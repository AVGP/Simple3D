Detailed Description

This class is responsible for z-buffering.

This class is the foundation of the z-buffering. Its used to determine, if a pixel should be drawn or not and is used in S3DLine and S3DPoint to check and update the z-buffer, before/after the actual drawing. Simple3D manages to keep the z-buffer clean and passes the reference to the entities.
